This is the JWT auth branch. There is a branch with session-based auth

# Usage

1. `npm install`

2. `cd server && npm install`

3. `cd ../client && npm install`

4.

```sql
CREATE DATABASE <database_name>;
```

5. fill `server/.env` and `client/.env` by `.../.env.example`
